# Grafické podklady — instrukce pro grafiky

Tato složka obsahuje popisy a zadání pro všechny vizuální prvky knihy.
Hotové soubory (PNG, SVG, AI) sem necommitujte — jsou příliš velké pro Git.
Místo toho:

1. Pojmenujte soubor stejně jako tento `.md` popis, jen s příponou souboru
2. Nahrajte ho do sdíleného Dropboxu (odkaz viz Slack)
3. Do tohoto `.md` souboru přidejte poznámku `STATUS: hotovo` a odkaz
