# Průvodce moderním cestovatelstvím 🌍

**Pracovní draft knihy — verze 0.1**

Toto je společný projekt naší redakce. Kniha je rozdělena do kapitol, na kterých pracují různí autoři. 
Každý přispěvatel pracuje na svém okruhu a výsledky spojujeme dohromady.

## Struktura repozitáře

```
/
├── README.md              ← tento soubor
├── OBSAH.md               ← obsah a stav kapitol
├── AUTOŘI.md              ← přehled přispěvatelů
├── kapitoly/
│   ├── 01-uvod.md
│   ├── 02-planovani.md
│   ├── 03-doprava.md
│   ├── 04-ubytovani.md
│   ├── 05-jidlo.md
│   └── 06-bezpecnost.md
├── obrazky/
│   ├── README.md          ← instrukce pro grafiky
│   ├── obalka-navrh-v1.md ← popis návrhu obálky
│   └── mapa-sveta.md      ← popis mapové ilustrace
└── navrhy/
    └── barevna-paleta.md  ← návrh vizuálního stylu
```
